== Description ==
Ajax mini cart for woocommerce by Artem Koliada. Adds ajax mini cart to your site.

== Installation ==

1. Upload the entire `zt-ajax-mini-cart` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Ajax Mini Cart' menu in your WordPress admin panel ( Tools > Ajax Mini Cart ).
